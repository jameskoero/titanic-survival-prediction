# Visualizations Documentation

This document provides an overview of the visualizations generated during the titanic survival prediction analysis.

## Confusion Matrix
- File: `outputs/confusion_matrix.png`
- Description: Displays the performance of the classification model from the predictions made. Helps to visualize the true positives, true negatives, false positives, and false negatives.

## ROC Curve
- File: `outputs/roc_curve.png`
- Description: Represents the trade-off between the true positive rate and false positive rate by plotting these two values against each other. It helps in evaluating the performance of the classification model.

## Feature Distribution
- File: `outputs/feature_distribution.png`
- Description: Shows the distribution of the features used in the model. Helps in understanding the data and identifying any potential outliers.

## Feature Importance
- File: `outputs/feature_importance.png`
- Description: Illustrates the importance of each feature used in making predictions. This is crucial for identifying which features contribute the most to the model's prediction accuracy.

Please ensure the PNG files are generated and stored in the outputs directory accordingly.
