# 🚢 Titanic Survival Prediction

![Titanic Badge](https://img.shields.io/badge/Project-Titanic%20Survival%20Prediction-blue?style=for-the-badge&logo=ship)

## Overview

This project implements machine learning models to predict survival outcomes for passengers aboard the Titanic using the classic Titanic dataset.
